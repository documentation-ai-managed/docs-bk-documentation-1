## What is Bk?

Bk is a powerful SaaS platform that helps you reconnect with old friends, ex-classmates, neighbors, and colleagues. You can search by name, school, workplace, or location to find and message people from your past. With secure profiles, private messaging, and group events, Bk makes networking effortless and privacy-focused.

<Callout kind="info">
  Bk prioritizes user privacy—your data stays protected with end-to-end encryption.
</Callout>

## Common Use Cases

Tailor Bk to your needs across different scenarios.

<Tabs>
  <Tab title="Class Reunions" icon="users">
    Search by graduation year and school to reconnect with classmates.
    
    ```javascript
    // Example search query
    const results = await bk.search({
      school: "Lincoln High",
      year: 2010
    });
    ```
  </Tab>
  <Tab title="Work Colleagues" icon="briefcase">
    Filter by company and department for professional networking.
    
    ```javascript
    const colleagues = await bk.search({
      company: "TechCorp",
      department: "Engineering"
    });
    ```
  </Tab>
  <Tab title="Neighborhood" icon="home">
    Locate neighbors by address or zip code.
    
    ```javascript
    const neighbors = await bk.search({
      location: "123 Main St, Anytown"
    });
    ```
  </Tab>
</Tabs>

## Getting Started

Follow these steps to begin using Bk.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your free account at bk.com with email or social login.
  </Step>
  <Step title="Build Profile" icon="edit-3">
    Add your schools, workplaces, and locations to improve search matches.
  </Step>
  <Step title="Start Searching" icon="search">
    Use the search bar to find people—results appear instantly.
  </Step>
  <Step title="Connect" icon="message-circle">
    Send a message or join a group event to reconnect.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Initialize Bk SDK
import { Bk } from '@bk/sdk';

const bk = new Bk({ apiKey: 'your-api-key' });

const profile = await bk.createProfile({
  name: 'John Doe',
  schools: ['Lincoln High']
});
```
```python
# Initialize Bk SDK
from bk import Bk

bk = Bk(api_key='your-api-key')

profile = bk.create_profile(
    name='John Doe',
    schools=['Lincoln High']
)
```
</CodeGroup>
