## Overview

Stay informed about the latest enhancements to Bk Documentation, your platform for reconnecting with old friends, ex-classmates, neighbors, and colleagues. Review changes below to understand new features, bug fixes, and improvements. You access updates directly in your dashboard under `Settings > Updates`.

<Callout kind="info">
  Breaking changes appear with `tags={["breaking"]}`. Review migration steps before upgrading.
</Callout>

## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search filters for location and shared interests
- Introduced real-time notifications for new connection requests

## Improvements

- Optimized profile loading speed by `>40%`
- Enhanced mobile responsiveness for group chats

## Bug Fixes

- Fixed duplicate contact suggestions
- Resolved login issues on Safari browsers

```javascript
// Example: Use new search API
const searchConnections = async (query) => {
  const response = await fetch('/api/connections/search', {
    method: 'POST',
    body: JSON.stringify({ query, filters: { location: 'New York' } }),
  });
  return response.json();
};
```

</Update>
